## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Version control system (Git preferred)
- Text Editor (VS Code preferred)

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/hmunish/capstone-htmlcssjs.git
```

### Install

Install this project with:

- Clone the repository in your local folder.

### Usage

To run the project, execute the following command:

- Run index.html file of the cloned repository in your browser.

### Deployment

- You can upload the cloned repository to your live server.
